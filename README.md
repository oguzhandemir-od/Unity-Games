# 🎮 Unity Games - Oyun Geliştirme Kütüphanesi

Bu depo; Unity Game Engine ve C# kullanarak 2D, 3D ve farklı oyun türlerinde (Hyper-Casual, Arcade, Mechanics vb.) geliştirdiğim oyun projelerimi, prototiplerimi ve eğitim çıktılarımı tek bir çatı altında toplayan **ana oyun geliştirme portfolyomdur**.

Büyük boyutlu Unity asset'lerinin ve proje geçmişlerinin karmaşa yaratmasını önlemek, geliştirme sürecimi merkezi bir yapıda tutmak amacıyla **Monorepo** mimarisiyle tasarlanmıştır. Projede gereksiz dosya kalabalığını önlemek için optimize edilmiş bir `.gitignore` yapısı kullanılmaktadır.

## 📂 Kütüphane Yapısı

Kütüphane içerisindeki klasörlere dair bilgi:

* [BTK-Akademi](/BTK-Akademi) - BTK Akademi tarafından düzenlenen kurs ve atölyelerde geliştirdiğim Unity oyun projeleri.

* [Gazi-FOW](/Gazi-FOW) - Gazi Üniversitesi FOW programında sektör standartlarına uygun oyun mekanikleri ve sahne tasarımları üzerinde çalıştığım teknik uygulamalar.

* [TechCareer](/TechCareer) - Techcareer bootcamp ve eğitim süreçlerinde geliştirdiğim Unity oyun projeleri.

## 🛠️ Kullanılan Teknolojiler & Yetkinlikler

* **Oyun Motoru:** Unity (2D & 3D)
* **Programlama Dili:** C# (Object-Oriented Programming for Games)
* **Unity Sistemleri:** Unity Physics (RigidBody, Colliders), UI Toolkit / Canvas, Prefab Yapıları, Particle Systems, NavMesh (Yapay Zeka Navigasyonu)
* **Mimari:** State Pattern, Singleton Pattern gibi oyun geliştirmede sık kullanılan tasarım desenleri.
